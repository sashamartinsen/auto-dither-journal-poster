# Dither ASCII Poster Tool

Static HTML/CSS/JS tool for GitHub Pages.

## v16 changes
- Independent size sliders for ASCII layers:
  - Words size
  - Glyph size
  - Grid size
- Words and glyph particles are placed by cell anchors and aligned to the left/top of their cells instead of centered.
- Words layer now uses a simple internal layout grid with configurable row count.
- Previous v15 compact UI and primary/secondary image layer structure preserved.
