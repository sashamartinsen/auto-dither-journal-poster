# Dither ASCII Poster Tool

Static HTML/CSS/JS tool for GitHub Pages.

## Features
- Base image dither layer with full controls
- Optional second image dither layer with its own controls
- Per-layer image positioning: image scale, offset X, and offset Y
- ASCII Graphics split into three togglable layers:
  - symbol grid (bottom)
  - glyph particles on a design grid (middle)
  - words (top)
- Glyph particles use configurable grid rows/columns for a stronger rhythmic layout
- Grid layer can place a chosen repeated symbol or random symbols across the whole canvas
- Random all colors button for saturated color palettes
- Export final PNG
